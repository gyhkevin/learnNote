### 我们的建议
通过这个项目，你会对数据分析的 核心步骤 有充分的了解，并帮助你了解在未来课程中将会学到的知识。这个项目和你未来所接触到的稍有不同，它并 不需要 你提前观看任何课程视频，它的目的是帮助你设置基础的学习环境，了解数据分析的吸引力，以及这门课程是否真的适合你。此外，你还会通过此项目熟悉优达学城的 项目审核及反馈流程 。因此，我们建议你在加入纳米学位的 前7天 内完成此项目。
这个项目需要学生了解基础的 Python 编程知识。如果你之前没有用使用过，我们建议你可以先学习优达学城的 计算机科学导论的前4课。
我如何完成此项目？

1. 在浏览器中打开 http://continuum.io/downloads
2. 使用 Anaconda 发布版（自带 IPython Notebook）安装 Python。我们建议你按照此方式安装，因为 Anaconda 自带很多有用的包，安装非常方便。请勿安装 Python 3.4，因为纳米学位项目在整个教学和项目中都使用 Python 2.7。
	>- 在 PC 上，单击 Windows 图标后选择“Windows 64-Bit Python 2.7 Graphical Installer”。如果你的计算机为 32 位系统，你也可以选择 32 位的安装程序。运行安装程序并按屏幕上的指示操作。
	>- 在 Mac 或 Linux 上，按照同样的步骤进行，但是要选择合适的安装程序。
3. 打开此链接，下载项目文档。你也可以访问我们的 GitHub repo，下载数据分析师纳米学位的全部项目文档。
4. 打开命令提示符 (PC) 或终端（Mac 或 Linux）。
	>- 在 PC 上，单击“开始”按钮，然后搜索“命令提示符”。
	>- 在 Mac 上，同时按下 command 和空格键，然后在 Spotlight 搜索框中输入“终端”。你也可以在 Finder 中搜索“终端”。
5. 导航至你下载了 IPython Notebook 文件的目录。
	>- 在 PC 上，你可以输入：cd C:\Users\username\Downloads\，并将 username 替换为你的用户名。了解更多基础终端命令。
	>- 在 Mac 或 Linux 上，你可以输入：cd ~/Downloads（假如这是你的浏览器保存下载文件的默认地址）。了解更多基础终端命令。
6. 在终端中运行命令 jupyter notebook Bay_Area_Bike_Share_Analysis.ipynb。
7. 通读笔记本，然后回答问题。请记住，如果遇到困难或问题，你可以参考统计学课程、在 Google 上搜索或前往论坛求助。
8. 完成后，将笔记本以 HTML 格式导出。在 IPython Notebook 中，单击 File -> Download.As -> HTML (.html)。
	Jupyter Notebook 里有提交项目所需要的所有内容。建议你通读该文件，回答其中的问题并根据提示运行代码块。若在这个过程中你遇到任何问题，请不要犹豫在 论坛 发贴，将有专业的导师帮你回答。
